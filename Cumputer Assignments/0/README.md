# Ethereum Transaction Data Analysis(Web Scraping)

A data science project that explores Ethereum blockchain transactions through web scraping, statistical analysis, and sampling techniques.

## Overview

This project collects transaction data from the latest Ethereum blocks on Etherscan and performs exploratory data analysis on transaction values and fees. It also compares population statistics against different sampling methods.

## Features

- Web scraping with Selenium and BeautifulSoup
- Data cleaning and preprocessing using Pandas
- Statistical analysis of transaction values and fees
- Data visualization with Matplotlib and Seaborn
- Simple Random Sampling (SRS)
- Stratified Sampling
- Population vs. sample comparison

## Tech Stack

## Getting Started

Install the required dependencies:

```bash
pip install selenium beautifulsoup4 pandas numpy matplotlib seaborn
```

Run the Jupyter notebook:

```bash
jupyter notebook DS_CA0.ipynb
```
